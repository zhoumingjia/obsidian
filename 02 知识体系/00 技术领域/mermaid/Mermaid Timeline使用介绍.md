## 示例1：商务风多色分段项目时间线（适合汇报）

用 section 分组并自动配色，清晰区分阶段，搭配时长标注，专业感拉满。

```mermaid
timeline
    title 金融双录系统上线全流程（商务版）
    section 需求与设计 📝
    2026-01-01 : 需求调研与合规评审 : 5个工作日
    2026-01-07 : 系统架构与UI/UE设计 : 7个工作日
    section 研发与联调 🔧
    2026-01-17 : 核心功能开发 : 10个工作日
    2026-01-29 : AnyChat接口联调与适配 : 3个工作日
    section 测试与上线 🚀
    2026-02-02 : 功能/压力/兼容性测试 : 8个工作日
    2026-02-12 : 生产环境部署与切换 : 2个工作日
    2026-02-14 : 上线后7x24小时监控 : 持续
```

```mermaid
    \n1. 掌握 Nginx 反向代理与负载均衡基础配置

    2025-04 : ⚙️ 进阶调优

    \n1. 解决生产环境 Nginx 502/504 异常
    \n2. 排查 Tomcat 内存泄漏与 Full GC 频繁问题
```

## 精美好用的核心技巧

1.  **配色与分组**：用 section 划分模块，Mermaid 会自动分配不同颜色，最多支持 12 种差异化配色。
2.  **换行与排版**：子任务用 `\n` 强制换行，单个描述控制在 20-30 字，避免横向溢出。
3.  **图标点缀**：添加 emoji（📝/🔧/🚀 等）提升可读性，适配 Typora 亮/暗主题。
4.  **版本与设置**：Typora 需 ≥1.12.1，在「设置→Markdown→扩展语法」中勾选「图表」。
